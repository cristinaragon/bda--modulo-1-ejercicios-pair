# Ejercicios de Python – Módulo 1

Este repositorio contiene ejercicios prácticos de Python correspondientes al **Módulo 1**. Los ejercicios están diseñados para reforzar conceptos fundamentales como listas, diccionarios, sets y estructuras de control (if, for, while).

## Estructura de ejercicios

- `ejerciciolistas.ipynb` – Ejercicios sobre **listas** y operaciones básicas.
- `ejercicios_diccionarios.ipynb` – Ejercicios sobre **diccionarios** y manipulación de datos.
- `ejercicios_sets.ipynb` – Ejercicios sobre **sets** y conjuntos en Python.
- `ejercicios_sentencias_control.ipynb` – Ejercicios sobre **sentencias de control**, bucles y condicionales.
